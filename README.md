## Prerequisites

Before you begin, ensure you have met the following requirements:

1. Java Development Kit (JDK) installed on your system whichever version you can use. In this project, the contributor uses Java JDK 21.0.4. [Click here to download](https://www.oracle.com/id/java/technologies/downloads/).
2.  Visual Studio Code for streamlining code editor with support for development operations like debugging, task running, and version control. [Click here to download](https://code.visualstudio.com/download).
3. Git for managing source code and repository. [Click here to download](https://git-scm.com/downloads).

## Installation

Follow these steps to set up the project on your local machine:

Open your local directory (e.g., \Documents) and clone the repository:

```bash
git clone <repository_url>
```

Navigate to the project directory:

```bash
cd <project_directory>
```

Compile the source code:

```bash
javac <java_file>.java
```

Run the source code:

```bash
java <class_name>
```
